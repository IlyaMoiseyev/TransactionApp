# TransactionApp

TransactionApp - Java console application designed to securely make financial transactions between accounts and automatically generate a transaction report.
The system provides control over the correctness of amounts, keeps a log of actions and provides a convenient mechanism for filtering the history by date.

## Main functions of the application:

+ automatic generation of source files
+ source file processing, transaction execution, and operation logging in multithreaded mode
+ making money transfers between accounts
+ verifying the correctness of the transaction amount
+ logging operations to a report file
+ generating a report for a given period

## How to launch the app:
 - run `main` method into ApplicationLaunchService.java at `src/service`
